# 마찰보상 체험 웹

정지마찰과 운동마찰의 차이를 체험하고, 마찰보상 적용 전후의 위치 오차를 비교할 수 있는 정적 웹사이트입니다.

## 포함 기능

- 모터/회전 시뮬레이션
- 마찰보상 ON/OFF
- 목표 각도, 제어 강도, 정지마찰, 운동마찰 조절
- 정지마찰과 운동마찰 차이 반영
- 목표 위치 / 실제 위치 / 오차 표시
- 다시 실행 / 초기화
- 기초 / 비교 / 심화 모드
- 실시간 그래프
- 속도 의존 마찰 옵션
- 위치 의존 마찰 옵션
- 보상값 직접 조절
- 자동 비교 결과 표시

## 실행 방법

브라우저에서 `index.html` 파일을 열면 바로 실행됩니다.

## GitHub Pages 배포 방법

1. GitHub 저장소를 새로 만듭니다.
2. 이 폴더 안의 파일을 저장소 루트에 업로드합니다.
3. GitHub 저장소의 `Settings > Pages`로 이동합니다.
4. `Deploy from a branch`를 선택합니다.
5. Branch를 `main`으로 선택하고 `/root`를 지정합니다.
6. 저장하면 몇 분 후 배포 주소가 생성됩니다.

## 파일 구성

- `index.html` : 페이지 구조
- `styles.css` : 미니멀 UI 스타일
- `app.js` : 시뮬레이션 로직과 그래프 렌더링

## 디자인 방향

- 화려한 효과 없이 단정한 카드형 구성
- 흰색/연회색 기반
- 포인트 컬러 1개만 사용
- GitHub Pages에서 바로 쓸 수 있는 정적 구조
