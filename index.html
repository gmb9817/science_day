실험 안내

왜 처음에 잘 안 움직일까

정지 상태에서는 정지마찰이 먼저 버티기 때문에 제어 입력이 어느 정도 커지기 전까지는 각도가 거의 변하지 않습니다. 정지마찰 값을 높이면 이 현상이 더 분명하게 나타납니다.

보상을 끄면 어떤 일이 생길까

목표 각도까지 가기 직전 제어 입력이 약해지면 정지마찰을 다시 넘지 못해 중간에서 멈출 수 있습니다. 이때 실제 각도와 목표 각도 사이에 오차가 남습니다.

보상을 켜면 무엇이 달라질까

멈추기 쉬운 구간에서 목표 방향으로 힘을 조금 더 보태 주기 때문에 다시 움직일 가능성이 커집니다. 같은 설정에서도 보상 ON이 OFF보다 목표 각도에 더 가깝게 도달하는지 확인할 수 있습니다.

슬라이더를 이렇게 보면 된다

제어 강도는 기본적으로 얼마나 세게 움직이게 할지, 정지마찰과 운동마찰은 출발 전후 저항 차이를, 보상값은 멈추려는 순간 얼마나 더 밀어줄지를 뜻합니다. 값을 바꾼 뒤 결과 카드의 실제 각도와 오차를 비교해 보세요.
